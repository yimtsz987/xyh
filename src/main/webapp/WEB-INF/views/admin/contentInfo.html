湖南城市学院教育发展基金会简介

简 介

湖南城市学院教育发展基金会于2007年5月成立。原始基金数额为人民币200万元，来源于湖南城市学院已接受的社会捐赠。本会的宗旨：在国家法律法规和政策范围内，广泛利用各种渠道吸纳筹措资金，全面支持和推动湖南城市学院的建设和发展。本会的登记管理机关为湖南省民政厅，业务主管单位为湖南省教育厅。

业务范围

（一）筹集基金：接受社会各界，包括机关、团体、企事业单位和个人的捐赠（含现钞和物资等）。

（二）管理基金：通过银行存款、购买并持有债券、拍卖物资等方式，科学、规范地运作基金，使基金实现保值增值和连续积累，滚动发展。

（三）使用基金：支持湖南城市学院的建设和发展项目；奖励为湖南城市学院建设与发展做出突出贡献的单位和个人；资助在校贫困学生。
